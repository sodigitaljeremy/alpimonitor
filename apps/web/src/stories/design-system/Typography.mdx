## Snippet mono en contexte

Le bloc code SPARQL de OWhyLindasSection met JetBrains Mono en scène — les
mots-clés en `text-primary`, les URI (IRI) aussi pour remonter au-dessus du
seuil AA contre le fond blanc.

## Principes

- **Une fonte par rôle**. Inter pour lire, JetBrains Mono pour lire _du code_
  ou un nombre. Jamais l'inverse.
- **Max deux poids visibles par bloc**. 400 régulier et 600 semibold suffisent
  dans 95% des cas ; 500 est réservé aux labels mono (tracking élargi).
- **Aucune italique** dans l'UI française. Les titres n'en abusent pas, les
  paragraphes non plus — l'italique est gardée pour les citations dans les
  ADR (hors scope UI).
